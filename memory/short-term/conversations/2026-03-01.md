**时间**: 01:05 - 01:18
**标签**: #系统设计 #记忆 #长期规划

### 需求
1. 全文本记录所有对话（备份）
2. 分类管理：任务、规则、知识
3. 分层：短期/长期记忆
4. 每日总结
5. 参考傅盛的分层记忆理念

### 改进方案: 分层记忆系统

```
Layer 3: 长期记忆 (long-term/)
  - 身份档案 (identity/)
  - 领域知识 (knowledge/)
  - 重要决策 (decisions/)
  
Layer 2: 短期记忆 (short-term/)
  - 进行中任务 (tasks/)
  - 近期对话 (conversations/)
  
Layer 1: 工作记忆 (working/)
  - 当前会话 (current-session.md)
  - 今日上下文
  
Layer 0: 原始记录 (raw/)
  - 完整备份（压缩存储）
```

### 记录规则
- **实时**: 逐条追加到工作记忆
- **会话结束**: 提炼到短期记忆
- **每日23:59**: 蒸馏到长期记忆
- **每周**: 归档过期短期记忆

### 关键特性
- 语义标签系统（便于检索）
- 自动提炼 + 人工确认（混合模式）
- 人名独立档案
- 决策记录（可追溯原因）
